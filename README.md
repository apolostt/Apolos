# Apolos

## Apolos Shield (Android)

Bezpečnostní Android aplikace (APK) pro ochranu telefonu — ochranná VPN (DNS
filtr / kill-switch / WireGuard), detekce kamery, mikrofonu a záznamu obrazovky
s okamžitým alarmem, hlídání chování aplikací a datového toku, podpis APK
v1/v2/v3/v4.

Zdrojový kód a návod na sestavení: [`android/`](android/README.md).
APK sestaví GitHub Actions workflow **Build Apolos Shield APK** (artefakt ke stažení).
